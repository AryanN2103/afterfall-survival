# AFTERFALL : SURVIVAL

AFTERFALL : SURVIVAL is a terminal-based zombie survival game created using Python. The player must survive for 15 days by exploring locations, collecting supplies, fighting zombies, and managing resources such as health, hunger, and stamina.

---

# FEATURES

- Interactive combat system
- Day and night cycle
- Random story events
- Inventory and loot system
- Multiple weapons and ammo types
- Save and load functionality
- Location-based exploration
- Randomized gameplay encounters

---

# HOW TO RUN THE GAME

## Step 1 — Install Python

Download Python from:

https://www.python.org/downloads/

IMPORTANT:
During installation, make sure to enable:

Add Python to PATH

---

## Step 2 — Download the Project

Download or clone the repository files.

Make sure the following files are inside the same folder:

afterfall.py

(Optional)
poster.png

---

## Step 3 — Open Terminal

Open Command Prompt / Terminal inside the project folder.

Example:

cd Desktop/Afterfall

---

## Step 4 — Run the Game

Run the following command:

python afterfall.py

If that does not work, try:

python3 afterfall.py

---

# GAME OBJECTIVE

Survive for 15 days while exploring dangerous locations and managing resources carefully. Zombies become stronger at night, so every decision matters.

---

# CONTROLS

1. Explore  
2. Rest  
3. Inventory  
4. Use Item  
5. Save Game  
6. Quit

---

# SAVE SYSTEM

To save progress during the game, select:

5. Save Game

The game will automatically create a file called:

savegame.json

This file stores all player progress including:
- Health
- Inventory
- Ammo
- Day and time
- Player statistics

To continue a saved game later, run the program again and choose:

Load save? (y/n)

Press:
y

to load the saved progress.
